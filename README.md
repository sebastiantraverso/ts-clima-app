## Aplicacion del Clima hecha en TypeScript

### **Instalaciones necesarias**
Node
https://nodejs.org/es/

TypeScript
http://www.typescriptlang.org/


### **ejecutar**
```
node install
```


### **Ejemplos**
node app -d 'cordoba argentina' 

node app -d 'new york'

node app -d 'london'

node app -d 'tokio'

node app -d 'boston'
